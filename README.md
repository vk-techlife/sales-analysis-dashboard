# Sales Analysis Dashboard

##  Project Overview

This project analyzes sales data across different cities and product categories using Excel.

##  Tools Used

* Microsoft Excel
* Pivot Tables
* Charts

##  Key Insights

* Mumbai has the highest total sales
* Bangalore has the lowest sales
* Electronics category generates the highest revenue
* Fashion category has the lowest contribution
